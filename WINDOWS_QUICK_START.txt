⚠️  常見問題排查
───────────────────────────────────────────────────────────────────────────

問題 1：'psycopg' module not found
解決：
    pip install psycopg[binary]

問題 2：PostgreSQL connection failed
解決：
    1. 檢查 PostgreSQL 是否正在運行（Windows 工作管理員 → 服務）
    2. 檢查連接參數是否正確：
       - Host: localhost 或 127.0.0.1
       - Port: 5432 (預設)
       - User: postgres
       - Password: postgres (或您設定的密碼)

問題 3：執行腳本出現編碼錯誤
解決：
    # 改成明確指定編碼
    python -u << 'EOF'
    # ... 上面的 Python 程式碼 ...
    EOF

問題 4：PowerShell 腳本執行權限不足
解決：
    # 允許執行本地腳本
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

問題 5：\"cannot find module 'sqlalchemy'\"
解決：
    pip install sqlalchemy
