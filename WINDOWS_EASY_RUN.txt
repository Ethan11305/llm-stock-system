❌ 如果出現錯誤

【錯誤 1：ModuleNotFoundError: No module named 'sqlalchemy'】

解決：確保虛擬環境已激活，然後執行：
    pip install sqlalchemy psycopg[binary]


【錯誤 2：could not connect to server】

解決：
    1. 檢查 PostgreSQL 是否運行中
    2. 檢查密碼是否為 'postgres'（或您設定的密碼）
    3. 檢查資料庫名稱是否為 'llm_stock'


【錯誤 3：FileNotFoundError: db/sql/004_embedding_enhancements.sql】

解決：確認目錄是否正確
    cd C:\path\to\LLM理財
    dir db\sql\


═══════════════════════════════════════════════════════════════════════════

✅ 成功後立即執行下一步

1️⃣ 預檢 backfill（不實際寫入）：

    python scripts/backfill_embeddings.py --dry-run


2️⃣ 正式回填（如預檢成功）：

    python scripts/backfill_embeddings.py --batch-size 50


3️⃣ 手工測試查詢場景
